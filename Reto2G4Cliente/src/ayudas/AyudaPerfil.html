Esta es la ventana de Perfil:

Para mayor información consultar el manual

Elementos de la Ventana:

  • Perfil: Datos de la cuenta del usuario
  • Nueva Contraseña: Campo para ingresar la nueva contraseña.
  • Repetir Contraseña: Campo para repetir la nueva contraseña.
  • Aceptar: Botón para confirmar el cambio de contraseña.

Recuerda que puedes utilizar los elementos de la barra de menú para acceder a otras funciones de la aplicación:

  • Inicio: Volver a la pantalla de inicio.
  • Menú: Acceder a diferentes secciones como Perfil, Productos, Eventos y Cerrar Sesión.
  • Ayuda: Acceder a la sección de ayuda.
